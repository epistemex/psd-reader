Compatibility Mode

The PSD file must be saved in compatibility mode which is the typical
save mode.

Photoshop® renders all layers, effects and masks into a single
bitmap which is stored inside the PSD file as a "flat" bitmap when saved.
This so that older application which does not support new features and ways
to mix layers, or where some fonts are missing, can still show the image
as it was intended.

This bitmap is also what we use as image source. If for some reason the
compatibility mode is turned off, this image is just a simple placeholder
image and all layers has to be rendered manually which out of scope for this
project, which sole purpose is to give a fast and simple way to display
PSD files.

We recommend therefor to always keep the default setting of the compatibility
option enabled in Photoshop®.
